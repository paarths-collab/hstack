# scripts/bump-versions.sh (a future utility) is a single-edit upgrade.
#
# Format: KEY=value, one per line. Blank lines and lines starting with `#`
# are ignored. Values are pure strings — no shell expansion. Keep them
# quotable-free (no `$`, `` ` ``, backslashes).
#
# When you bump a pin here, also:
#   1. Update the relevant SKILL.md files (grep for the old pin).
#   2. Add a CHANGELOG.md entry under "Changed".
#   3. Run scripts/validate-skills.sh — the validator asserts the pin is
#      referenced consistently and that no skill has drifted onto a
#      different version.
